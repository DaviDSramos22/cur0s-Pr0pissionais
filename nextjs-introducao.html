Introdução ao Next.js: React com Server-Side Rendering

Início Tutoriais Sobre Contato

Next.js é um framework React que traz renderização do lado servidor (SSR) e geração estática, melhorando performance e SEO das aplicações.

Por que usar Next.js?

  • Renderização no servidor para melhor performance e SEO.
  • Rotas automáticas baseadas na estrutura de pastas.
  • Suporte a API routes para criar backends simples.
  • Suporte a geração estática e incremental.

Iniciando um projeto Next.js

npx create-next-app@latest meu-projeto-next

Após a instalação, entre na pasta e execute:

npm run dev

Estrutura básica

  • pages/: Contém os arquivos de páginas. Cada arquivo representa uma rota.
  • public/: Arquivos estáticos (imagens, etc.).
  • components/: Componentes React reutilizáveis.

Exemplo simples de página

export default function Home() {
  return (
    <div>
      <h1>Bem-vindo ao Next.js!</h1>
      <p>Esta é a página inicial.</p>
    </div>
  );
}
