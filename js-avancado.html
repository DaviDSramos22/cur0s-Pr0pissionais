JavaScript Avançado: Promises, Async/Await e APIs

Início Tutoriais Sobre Contato

Neste tutorial, vamos aprofundar no uso de JavaScript moderno com Promises, Async/Await e o consumo de APIs REST, habilidades essenciais para construir aplicações web robustas e interativas.

Promises em JavaScript

Uma Promise representa uma operação assíncrona que pode ser concluída com sucesso ou rejeitada com um erro. Promises ajudam a evitar o "callback hell" e deixam o código mais legível. 


const minhaPromise = new Promise((resolve, reject) => {
    setTimeout(() => {
        const sucesso = true;
        if (sucesso) {
            resolve('Operação concluída com sucesso!');
        } else {
            reject('Erro na operação.');
        }
    }, 2000);
});

minhaPromise
    .then(resultado => console.log(resultado))
    .catch(erro => console.error(erro));

Async/Await

O Async/Await é uma sintaxe moderna que facilita o uso de Promises, tornando o código assíncrono parecido com código síncrono. 


async function exemploAsync() {
    try {
        const resultado = await minhaPromise;
        console.log(resultado);
    } catch (erro) {
        console.error(erro);
    }
}
exemploAsync();

Consumindo APIs REST com fetch

O método fetch() permite fazer requisições HTTP para APIs e obter dados em formato JSON, texto ou outros. 


fetch('https://jsonplaceholder.typicode.com/posts')
    .then(response => response.json())
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error('Erro ao buscar dados:', error);
    });

Exemplo com Async/Await e fetch


async function buscarPosts() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts');
        const posts = await response.json();
        console.log(posts);
    } catch (error) {
        console.error('Erro ao buscar posts:', error);
    }
}
buscarPosts();

Dicas finais

  • Sempre trate erros em requisições assíncronas.
  • Use Async/Await para código mais limpo e fácil de entender.
  • Explore APIs públicas para praticar consumo de dados.

Dominando essas técnicas, você estará pronto para construir aplicações web modernas, responsivas e com integração de dados em tempo real.
