JavaScript Avançado: Promises, Async/Await e APIs

Início Tutoriais Sobre Contato

Este tutorial aborda conceitos avançados de JavaScript, focando em programação assíncrona usando Promises, Async/Await e consumo de APIs REST.

Promises

Promises representam operações assíncronas que podem ser resolvidas (sucesso) ou rejeitadas (erro). Exemplo básico:


const promessa = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve("Sucesso!");
  }, 2000);
});

promessa.then(resultado => {
  console.log(resultado);
}).catch(erro => {
  console.error(erro);
});

Async/Await

Async/Await permite escrever código assíncrono de forma mais legível, parecido com código síncrono:


async function exemplo() {
  try {
    const resultado = await promessa;
    console.log(resultado);
  } catch (erro) {
    console.error(erro);
  }
}
exemplo();

Consumindo APIs REST

Exemplo usando fetch para obter dados de uma API:


fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Erro:", error));

Recomendações

  • Use try/catch para tratar erros com Async/Await.
  • Prefira Async/Await para código mais limpo.
  • Conheça bibliotecas como axios para requisições HTTP.
