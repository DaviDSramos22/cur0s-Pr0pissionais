Adicionando interatividade às suas páginas web

O que é JavaScript?

JavaScript é uma linguagem de programação que permite adicionar interatividade a páginas web. Com ele, você pode responder a eventos, validar formulários, atualizar conteúdos dinamicamente, criar animações, entre outras funcionalidades.

Como adicionar JavaScript a uma página?

Você pode inserir JavaScript diretamente no HTML, usando a tag <script>, ou referenciar arquivos externos.

<script>
    alert("Olá, mundo!");
</script>

Variáveis e Tipos de Dados

Use let, const ou var para declarar variáveis:

let nome = "João";
const idade = 25;
var ativo = true;

Funções

Funções são blocos de código reutilizáveis:

function saudacao(nome) {
    return "Olá, " + nome + "!";
}

console.log(saudacao("Maria"));

Eventos e DOM

Você pode capturar eventos do usuário e modificar elementos HTML com JavaScript.

document.getElementById("meuBotao").addEventListener("click", function() {
    alert("Botão clicado!");
});

Exercício Prático

Tente criar uma página com um botão que muda o texto de um parágrafo quando clicado.
