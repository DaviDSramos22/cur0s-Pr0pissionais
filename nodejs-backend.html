Backend com Node.js: Criando seu primeiro servidor

Início Tutoriais Sobre Contato

Node.js é uma plataforma que permite executar JavaScript no servidor, possibilitando construir aplicações backend escaláveis e eficientes.

🔧 Configurando o ambiente

Primeiro, instale o Node.js no seu computador pelo site oficial https://nodejs.org.

🛠️ Criando um servidor básico

Vamos criar um arquivo chamado server.js com o seguinte código: 


const http = require('http');

const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Olá, mundo! Este é meu servidor Node.js.');
});

server.listen(3000, () => {
    console.log('Servidor rodando em http://localhost:3000');
});

🚀 Executando o servidor

No terminal, navegue até a pasta do projeto e execute: 


node server.js

Acesse http://localhost:3000 no navegador para ver sua mensagem.

📦 Usando Express.js

Express.js é um framework que simplifica a criação de servidores web em Node.js. Instale com: 


npm install express

Exemplo básico com Express: 


const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send('Olá do Express!');
});

app.listen(3000, () => {
    console.log('Servidor Express rodando na porta 3000');
});

📚 Integração com Banco de Dados

Você pode conectar seu backend a bancos de dados para armazenar e consultar dados, usando bibliotecas específicas para MySQL, MongoDB, PostgreSQL, entre outros.

Esse tutorial oferece uma base para você criar servidores e APIs para suas aplicações.
